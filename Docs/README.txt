MOMENTUM — Bibliothèque SVG sports

Source graphique : Lucide Icons (licence ISC).
Les fichiers ont été renommés pour correspondre aux clés icon de momentum-sports.js.

Installation :
Copier le contenu de assets/icons/sports/ dans le même dossier de ton projet.
Conserver LUCIDE-LICENSE.txt dans le dépôt.

Important :
Certaines disciplines sans icône dédiée dans Lucide utilisent temporairement une icône générique proche.
Elles pourront être remplacées individuellement sans modifier momentum-sports.js ni momentum-icons.js.
